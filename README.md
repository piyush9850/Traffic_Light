Traffic Light Controller using Verilog HDL

A Finite State Machine (FSM) based Traffic Light Controller designed using Verilog HDL and implemented on FPGA using Vivado. This project demonstrates RTL design flow from coding and simulation to synthesis and implementation.

📌 Project Overview

This project simulates a two-road traffic light controller system where traffic signals transition automatically based on timing logic.

The design is implemented using:

FSM (Finite State Machine)
Sequential logic
Counters and timers
Verilog HDL
FPGA implementation flow in Vivado

The project helped in understanding how RTL code gets converted into actual FPGA hardware using LUTs and Flip-Flops.

🚦 Features
FSM based traffic control
Clock driven state transitions
Red, Yellow, Green sequencing
Parameterized timing design
RTL simulation and waveform verification
Synthesizable FPGA design
Vivado implementation flow
